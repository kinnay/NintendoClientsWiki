## README

This repository contains a mirror of https://github.com/kinnay/NintendoClients/wiki. It is automatically synchronized with the wiki, in both directions.

This repository was created so that pull requests can be submitted for the wiki. If you would like to suggest changes to the wiki, please open a pull request on this repository.

Contributors with a proven track record may be given write access to this repository directly. Send me a message on Discord if you would like to request such access.

If you want to link to a specific page of the wiki, please link to the [original wiki](https://github.com/kinnay/NintendoClients/wiki) because it provides a better UI than this repository.
